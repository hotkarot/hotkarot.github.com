OpenSauce by HotKarot

Social cooking with GitHub

The idea of the open sauce project is to get people to publish recipes on GitHub so that they can improve them in collaboration.

So start being social, follow HotKarot or watch open sauces to see what's happening. Do you see something wrong with a recipe? Too much salt? Raise an issue. Would you suggest an improvement? Recommend a special spice? Fork a sauce! Do you think you can prepare a better sauce? Let us know, send us a pull request!

The GitHub repository for this project can be found at: https://github.com/hotkarot/hotkarot.github.com
